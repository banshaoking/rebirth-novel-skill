# 重生流都市小说复现 Skill

> 基于柳岸花又明《我真没想重生啊》《重生高考前99天》两部作品提炼的写作范式

---

## 一、核心设定模板

### 1. 主人公设定公式

```
【前世身份】：{年龄}岁的{职业}，{成功但有核心遗憾的描述}
【重生触发】：{意外事件}（车祸/猝死/其他），时间线倒回{具体年份}
【重生节点】：{人生关键转折点}（高考前/毕业季/大学入学）
【核心矛盾】：前世的{遗憾} vs 重生后的机会
【性格底色】：前世{性格缺陷}，重生后因{阅历加持}而{反差表现}
```

**关键原则**：
- 前世必须"成功但不幸福"——有钱没爱情，或有地位没自由
- 重生节点必须是"选择分岔口"——高考填志愿、毕业告白、人生方向抉择
- 性格反差是**喜剧核心**：成年人的社交习惯在少年身体里形成"肌肉反应"

### 2. 女性角色矩阵（至少2人，形成张力）

| 角色类型 | 代号 | 功能 | 示例 |
|---------|------|------|------|
| **白月光** | 可望不可即型 | 前世错过的遗憾，重生后核心目标 | 萧容鱼、宋时微 |
| **宝藏女孩** | 默默付出型 | 与白月光形成对比，制造选择困境 | 沈幼楚、俞弦 |
| **搅局者**（可选） | 活泼/反叛型 | 打破稳态，推动情节 | 罗璇 |

**关键原则**：
- 白月光 = 前世的"如果当初"；宝藏女孩 = 今生的"原来一直在"
- 两条感情线必须都有充分的理由让读者支持
- 最终选择要出人意料但情理之中

## 二、开篇结构模板（第1-3章）

### 第1章：前世终局 + 重生触发

```
场景A：成年主角的"高光但孤独"时刻
  - 饭局/会议/独处 → 展示社会地位
  - 一个刺痛的细节（相亲对象的嘲讽/老同学的疏远/深夜的空虚）
  - 内心独白：如果当初……

场景B：意外触发
  - 车祸/意外 → 短暂的意识模糊
  - 感官切换：从现代都市 → 年少时的声音/气味/触感

场景C：醒来
  - 第一人称视角发现身体变小
  - 一个标志性细节确认年代（流行歌曲/电视节目/手机型号）
  - 结尾hook：一个成年人的"职业病"在少年身体里的荒诞反应
```

**《99天》范例**：陈着对枯死的发财树本能地恭维——"体制内的肌肉反应"
**《没想重生》范例**：陈汉升在公交车上被王梓博摇醒，街边放着F4的《流星雨》

### 第2章：能力确认 + 时代锚定

```
- 主角测试自己保留了什么能力（解题能力/社交直觉/商业嗅觉）
- 通过配角对话交代当前时间线的关键信息
- 发现"金手指"：不是超能力，而是成年人的判断力 + 对未来的模糊记忆
- 第一个蝴蝶效应：改变了一个小决定
```

### 第3章：核心悬念确立

```
- 白月光角色正式登场（或通过他人提及）
- 宝藏女孩角色初露端倪
- 明确主角的"重生任务"（不是拯救世界，而是弥补个人遗憾）
- 第一个"成年人视角碾压同龄人"的名场面
```

## 三、情节推进节奏

### 卷结构（每卷约30-50章）

```
第一卷：【校园篇】—— 重生初期适应 + 高考/校园核心事件
  - 3-5个校园事件（考试、运动会、同学冲突等）
  - 每个事件都展示"成年智慧碾压"
  - 感情线萌芽
  - 卷末高潮：高考/重大选择

第二卷：【转折篇】—— 进入新阶段（大学/创业）
  - 环境升级，人物关系扩展
  - 事业线启动
  - 感情线升温 + 第一次真正的选择困境
  - 卷末高潮：事业突破或感情转折

第三卷+：【展开篇】—— 多线并行
  - 事业线（创业/投资/博弈）
  - 感情线（三角/多角关系的拉扯）
  - 人际线（前世遗憾的人物重新登场）
  - 每50章一个大高潮
```

### 节奏公式

```
日常轻松（3-5章）→ 事件升级（2-3章）→ 高潮冲突（1-2章）→ 收获/奖励（1章）→ 新日常
```

## 四、喜剧风格要素

### 1. "肌肉反应"喜剧
主角因前世职业习惯，在少年身体里做出不合时宜的举动：
- 公务员习惯性恭维
- 商人习惯性算账
- 成年人的社交客套在同学间显得格格不入

### 2. 信息差幽默
主角知道未来但不能说，制造反讽：
- 看到同学追捧的东西，内心知道它很快会过时
- 对"预言"式建议无法解释理由

### 3. 身份错位
- 老师面前：身体是学生，灵魂是成年人，对话产生错位
- 同龄人面前：过于成熟的言行引起疑惑
- 父母面前：突然变得懂事，父母反而担心

---

## 五、时代感构建清单

每个重生流小说必须锚定具体年份，并填充以下细节：

| 类别 | 需要填充的细节 |
|------|--------------|
| **音乐** | 当年最火的歌（用于场景BGM） |
| **影视** | 当年热播剧/电影 |
| **科技** | 手机型号、电脑配置、网络状态 |
| **物价** | 房价、工资水平、消费水平 |
| **事件** | 可以"预测"的真实大事件（但不要直接写，用暗示） |
| **流行语** | 当年校园里的口头禅 |
| **校服/文具** | 时代特征的物质细节 |

## 六、人物对话风格

### 主角对话原则
- **对同龄人**：偶尔冒出成年人的用词，但会及时收回
- **对长辈/老师**：过于得体，反而显得老成
- **内心独白**：用30+岁的心态吐槽18岁的环境
- **关键时刻**：真情流露时回归少年感

### 配角对话原则
- 铁哥们：插科打诨，提供喜剧缓冲
- 白月光：初期内敛/高冷，逐步展露真实性格
- 宝藏女孩：温暖朴素，台词量随剧情递增
- 反派/对手：有明确的行为动机，不脸谱化

---

## 七、感情线推进节奏

```
Phase 1【距离感】（1-30章）
  白月光：高不可攀，主角只能远观
  宝藏女孩：擦肩而过，尚未注意到

Phase 2【接触期】（30-80章）
  白月光：因某个事件产生第一次正式交流
  宝藏女孩：因日常接触逐渐熟悉

Phase 3【拉扯期】（80-150章）
  白月光：关系升温但有外部阻力
  宝藏女孩：暗恋/默默付出被主角察觉
  核心张力：两个都有理由，读者分成两派

Phase 4【抉择期】（150-250章）
  某个关键事件迫使主角做出选择
  选择后的情感处理要细腻，不能轻描淡写
  被放弃的一方要有体面的退场

Phase 5【收束期】（250章+）
  确定关系后的甜蜜日常
  事业与感情的最终平衡
  回扣前世遗憾——"这次我没有错过"
```

---

## 八、避免的雷区

| 雷区 | 说明 |
|------|------|
| **主角全知全能** | 重生不是开挂，要有信息模糊和判断失误 |
| **女性角色工具化** | 每个女性角色都要有独立的目标和成长弧 |
| **感情线拖沓** | 选择不能无限拖延，读者会疲劳 |
| **事业线太爽** | 太顺利的事业线缺乏张力，需要重大挫折 |
| **时代感缺失** | 不标注年份的重生流没有灵魂 |
| **前世回忆过多** | 前世是调味料，不是主菜 |
| **配角纸片化** | 铁哥们、父母、老师都需要有立体感 |

## 九、写作启动清单

在开始写之前，确认以下所有项目：

- [ ] 主人公前世身份、职业、年龄、核心遗憾
- [ ] 重生触发事件的具体场景
- [ ] 重生的具体年份和节点
- [ ] 白月光角色的完整人设（外貌/性格/家庭/目标）
- [ ] 宝藏女孩角色的完整人设
- [ ] 铁哥们角色（提供喜剧和信息传递功能）
- [ ] 时代细节清单（音乐/影视/科技/物价）
- [ ] 第一卷的核心事件（3-5个）
- [ ] 感情线的最终走向（提前决定，但不要提前透露）
- [ ] 事业线的大致方向
- [ ] 全书预估章节数和卷数

---

## 十、一句话总结这个类型的本质

> **重生流的核心不是"知道未来所以赢"，而是"经历过遗憾所以珍惜"。**
> 喜剧是外壳，遗憾是内核，选择是驱动力。
