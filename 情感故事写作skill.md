# Craft Principles

## Ethical Boundary

This skill extracts craft-level practices: rhythm, scene selection, narrative architecture, comic distance, and reflective movement. It must not generate direct pastiche of any living author's protected style. If asked to imitate exactly, redirect to "your own voice with selected craft traits."

## Zhang Jiajia-Inspired Craft Moves

Use these as structural and rhythmic tools:

- Start with a scene that feels casual, then reveal it was emotionally loaded.
- Use simple spoken narration to lower the guard before a sharp feeling arrives.
- Let love, friendship, loss, and leaving appear through errands, meals, roads, messages, weather, and missed appointments.
- Build chapters/fragments as independent emotional episodes that can still echo a larger life arc.
- Delay the direct statement of pain. First show behavior, then memory, then the sentence that names it.
- Keep the pace cinematic: quick cuts, concrete locations, and a final image that feels like a camera holding after someone leaves.

Avoid:

- Overusing tearful aphorisms.
- Explaining every emotion immediately.
- Turning every paragraph into a quote-ready sentence.

## Wang Xiaobo-Inspired Craft Moves

Use these as intellectual and tonal tools:

- Let the narrator sound plain, rational, and slightly mischievous while discussing unreasonable things.
- Treat solemn topics with comic distance so the deeper critique lands more cleanly.
- Use logical escalation: begin with a small assumption, follow it too honestly, and expose absurdity.
- Allow the narrator to be clever but not pompous; the intelligence should feel like play.
- Mix desire, freedom, rules, embarrassment, and private thought without moral lecturing.
- Let narrative digressions become part of the structure: each detour should return with extra pressure or irony.

Avoid:

- Copying signature sentence shapes or famous argumentative cadences.
- Making every line sarcastic.
- Letting ideas flatten characters into mouthpieces.

## Hybrid Voice Formula

Use this blend when the user wants "my own style":

- **Surface**: clean, conversational Chinese.
- **Engine**: story first, thought second.
- **Temperature**: warm but not sugary; clever but not cold.
- **Movement**: incident -> joke/aside -> memory -> contradiction -> emotional or philosophical turn.
- **Ending**: neither over-explain nor vanish. Leave one precise image or one sentence of earned clarity.

## Practical Templates

### Opening Template

Begin with a concrete ordinary fact, then make it unstable. Treat this as a shape, not fixed wording:

1. Ordinary action: "That day, I was doing a small, specific thing."
2. Reframe: "Only later did I realize the small thing was carrying the real subject."
3. Pressure: add one object, one person, or one delayed message that changes the emotional weather.

### Paragraph Rhythm

Use paragraph lengths as pacing:

- 1-2 sentences: turn, joke, wound, reveal.
- 3-5 sentences: scene, memory, explanation.
- 6+ sentences: only when building a comic argument or sustained reflection.

### Emotional Landing

Choose one:

- Return to the first object with changed meaning.
- Let the narrator perform a small action instead of declaring growth.
- State a clear thought after enough scene evidence.
- End on contradiction: what was lost and what stayed.

## Output Defaults

When drafting:

- Offer one polished version first.
- If useful, add a short note naming the chosen structure and revision knobs.
- Keep explanations shorter than the prose unless the user asked for analysis.

When rewriting:

- Preserve the user's facts, emotional intention, and best original lines.
- Strengthen rhythm and transitions.
- Remove generic sentiment, decorative metaphors, and borrowed-sounding cleverness.
